Typography

Headings

For the website headings, I chose a Google font called Boogaloo. It is a decorative font and contrasts well with the body text font.

For the webpage headings, I used a bold font from Google Fonts called Raleway.

Navigation

For the navigation bar, I chose another Google font called Raleway. It's a sans-serif font that is easy to read.

Body Text

I chose to use the same Raleway font to try to keep the page consistent and simple, so the viewer can read easily.
